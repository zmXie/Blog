    Vue配置Sass全局混入

    xzm 5/6/2021 Vue
    1. 安装 sass-resources-loader 插件 
    npm i sass-resources-loader --save-dev
    1
    1. 配置 webpack.conf.js
    module: {
  rules: [
    {
      test: /\.(sc|sa|c)ss$/,
      use: [
        {
          loader: "sass-resources-loader",
          options: {
            resources: path.resolve(
              __dirname,
              "../src/assets/styles/mixins/index.scss"
            ),
          },
        },
      ],
    },
  ];
}
